# vue-watermark

>一个canvas制作背景水印，完全自定义。

## Build Setup

``` bash
# 安装组件
npm install vue-water-mark-tools --save

# 启动本地项目
npm run dev


# 使用详情
一.在main.js
1. import VueWater from 'vue-water-mark-tools/src/lib/index';
2. Vue.use(VueWater);

二.在使用的页面
<vue-water-mark-tools fatherId="hello"></vue-water-mark-tools>
